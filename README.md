# Badriville Resort - Next.js Migration

This repository contains the migrated Next.js application for Badriville Resort.

## Structure

- `src/app`: Contains the App Router pages.
- `src/components`: Contains the React components.
- `public/assets`: Contains the static assets (images).
- `legacy-angular`: Contains the original Angular application for reference.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Design

The design has been updated to be Classical, Premium, and Minimal, inspired by https://solomin-hotel.ru/en/.
It uses Tailwind CSS for styling and Playfair Display / Lato fonts.
